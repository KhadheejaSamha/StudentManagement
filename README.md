# Student Management System (Java)

A simple Student Management System built using Java.

## Features

* Add Student
* View Students
* Save Student Data to File
* Read Student Data from File

## Technologies Used

* Java
* File Handling
* Scanner Class
* VS Code
* Git & GitHub

## How to Run

Compile:

javac StudentFileManagement.java

Run:

java StudentFileManagement
